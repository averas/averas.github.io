üïπ GETTING TO KNOW THE DATASET

The dataset represents roughly 4000 office locations in the greater Stockholm area. Each datapoint is one office location with some attributes describing the characteristics of the office. The black/purple markers are Tele2 network hub sites, where we can connect customers. Your task will be to determine how we should prioritise the offices for marketing, based on their attributes. üíµ COMPANY REVENUE

All offices in the dataset belong to a company with a certain yearly income, or revenue. Arguably, companies with higher revenue may generate more business for us. üí∏ LEASED LINE COST

The black/purple markers on the map represent network hub sites where we can connect customers with leased lines. The cost of leased lines goes up with distance, so we might prefer connecting offices that are close to our sites. Select the COST mode in the visualisation and get a feel for how leased line cost goes up with distance to the nearest network site.

üè¢ NUMBER OF OFFICES

Although each data point only represents a single office, there is information on the data points on how many offices the parent company has in total. An important business opportunity for communication providers like Tele2 is interconnecting offices with each other, therefore companies with many offices may be more attractive customers than companies with fewer offices.

üëÄ OFFICE CLUSTERS

As mentioned above, leased line costs between our network hubs and customer offices impact the business case for any single office significantly. One way of approaching this is to lease lines between local customer deployments, which can be much shorter distances. This means that "clusters" of offices with many nearby offices (regardless of company) may be more attractive than isolated offices.

/* We only work with sustainble companies! */
var scoreFunction = (office) => {
    if (office.properties.sustainability < 90)
        return 0;
    return office.properties.revenue / 10; /* Scale to 0-100 */
};

/* Find offices with high revenue and low cost! */
var scoreFunction = (office) => {
    let score = office.properties.revenue / 20; /* Scale to 0-50 */
    score += 50 - Math.min(office.properties.cost, 50)
    return score;
};

/* Find clustered offices with high equality score! */
var scoreFunction = (office) => {
    let score = office.properties.nearby * 0.5; /* Scale to 0-25 */
    score += office.properties.equality * 0.75; /* Scale to 0-75 */
    return score;
};
